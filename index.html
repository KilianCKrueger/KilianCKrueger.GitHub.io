üåç Name The Domain v1

While playing GeoGuessr, I discovered that I couldn't reliably identify domains I came across. Motivated to memorize the domains, I created an app to practise and quiz me. It has two Game Modes:

  • 4 Answers - the question is displayed on top and the user selects the fitting answer from 4 buttons
  • Typing - the user uses a TextField input to type the correct answer; there is a hint button that gives the user the first 3 letters, then fills in the correct answer

Further, there is a switch to change whether domains or countries are asked for.

üåç Name The Domain v2

Name The Domain had a lot of potential, so I added

  • a button (and list) to switch between questions about capitals and domains
  • a difficulty system - for the hard mode my algorithm finds out how many entries in the answer list start with the same letter as the correct answer, then picks randomly from those with the same letter
  • a box showing the user's streak of correct answers
  • a help page to explain the app

Additionally, I overhauled the GUI to look more modern; cleaned up, refactored and encapsulated the code. Get it on GitHub.
